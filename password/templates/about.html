This project is a password manager application built using Django. The application is designed to securely store user passwords and login credentials in an encrypted form to ensure data protection and privacy. Features - User Authentication: Signup and Login: Users can create an account and log in securely. Both the signup and login processes ensure that user credentials are handled safely. Password Encryption: User passwords are hashed and stored securely using Django’s built-in authentication system, which employs industry-standard hashing algorithms. Password Storage: Encrypted Passwords: The app allows users to store various passwords (e.g., for websites, applications) in an encrypted format. This ensures that even if the database is compromised, the passwords remain secure. Fernet Encryption: Uses the cryptography library's Fernet symmetric encryption to encrypt and decrypt user passwords before storing them in the database. Security: Key Management: Encryption keys are securely managed and stored in a file that is excluded from version control using .gitignore, preventing unauthorized access. CSRF Protection: Implements Django’s Cross-Site Request Forgery (CSRF) protection to safeguard against CSRF attacks.
